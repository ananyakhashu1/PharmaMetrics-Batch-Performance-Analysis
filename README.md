# PharmaMetrics-Batch-Performance-Analysis
PharmaMetrics: Batch Performance Analysis

## Project Overview
Excel-based Pharmaceutical Manufacturing Analytics Dashboard for monitoring batch performance, profitability, and production KPIs.

## Features
- KPI Dashboard
- Top 5 Most Profitable Drugs Analysis
- Batch Approval vs Rejection Analysis
- Interactive Batch Search Tool
- Profitability Visualization

## Excel Functions Used
- SUM
- AVERAGE
- COUNT
- MAX
- IF
- AND
- INDEX
- MATCH
- IFERROR
- MID
- FIND
- LARGE
- COUNTIF
